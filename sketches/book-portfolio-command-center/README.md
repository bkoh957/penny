## Variant: Command Center

### Design stance
A dark, executive dashboard for deciding what deserves attention across five series.

### Key choices
- Layout: left rail with focus card, main matrix plus queue.
- Typography: dense system font with strong numeric tiles.
- Color: dark navy with production-stage chips.
- Interaction: active-only filter and Book 1 drilldown modal.

### Trade-offs
- Strong at: quick portfolio scan, blockers, next action.
- Weak at: detailed per-chapter editing; that belongs in a drilldown.

### Best for
A daily/weekly publishing cockpit where the top question is: what moves a book closer to release?
