## Variant: Minimal Command Ledger

### Design stance
Command-center information architecture with the warm editorial/paper style of the ledger, compressed into a minimalist bento-like operating screen.

### Key choices
- Series rows act as compact release ladders.
- Each book card uses a tiny bottom progress stripe for pipeline stage.
- Active Book 1 card embeds a 9-cell chapter sparkline directly in the matrix.
- Book drilldown modal shows all 27 chapters as a micro-grid.
- Queue is reduced to four terse next-action cards.

### Trade-offs
- Strong at: compressed at-a-glance status without spreadsheet clutter.
- Weak at: not enough room for detailed metadata; needs drilldowns for real work.

### Best for
The homepage of the publishing tracker: daily orientation, next action, and quick portfolio health.
