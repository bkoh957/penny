## Variant: Editorial Ledger

### Design stance
A warm, publishing-ledger view that treats each series as a release ladder rather than a generic task list.

### Key choices
- Layout: editorial document page with release ladders and chapter grid.
- Typography: serif masthead for strategy, sans-serif operational details.
- Color: paper, ink, amber, and status chips.
- Interaction: filters by active/parked series, chapter grid drilldown.

### Trade-offs
- Strong at: seeing each series as a product ladder with special novellas/reader magnets.
- Weak at: dense operations; less efficient than the command-center version for daily triage.

### Best for
Weekly planning, strategic review, and avoiding shiny-object drift across multiple possible series.
