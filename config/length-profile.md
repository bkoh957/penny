# Length Profile — word-count targets

```yaml
book_target_words: 65000
book_chapter_count: 24
```

## Chapter-type word-count ranges

| Chapter type | Word range |
|---|---|
| Opening chapter | 1,800–2,400 |
| Standard investigation / character chapter | 2,000–2,500 |
| Quick discovery / confrontation | 1,500–2,000 |
| Major reveal / emotional shift | 2,500–3,200 |
| Final confrontation chapter | 3,000–4,000 |

**Default** (type unclear from brief): 2,000–2,500 words.

A chapter **must not be submitted under its range minimum.** If prose runs short, extend a scene — deepen interiority, slow a beat, add sensory texture — rather than padding with recap or summary.
